# HN

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

HN is an alternative frontend to YCombinator's [Hacker News](https://news.ycombinator.com/). It's lightweight, mobile friendly, has a dark theme and doesn't send JavaScript to the client.

The main instance can be found [here](https://hn.ggtn.ch).
